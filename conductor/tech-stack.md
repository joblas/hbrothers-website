# Tech Stack: H Brothers Website Redesign

## 1. Frontend Core
*   **React (v19):** High-performance UI library for building the single-page application.
*   **TypeScript:** Type-safe development to ensure code quality and easier maintenance.
*   **Vite:** Ultra-fast build tool and development server for modern web development.

## 2. Artificial Intelligence
*   **Google Gemini AI:** Utilizing the `@google/generative-ai` SDK to power the H Brothers Concierge.
*   **Model:** `gemini-1.5-flash` for high-speed, cost-effective AI interactions.

## 3. Styling & UX
*   **Tailwind CSS:** Utility-first CSS framework for rapid, responsive design and ADA-compliant accessibility.
*   **React Router (v7):** For clean, professional URL handling and navigation.

## 4. Deployment & DevOps
*   **GitHub Pages:** Reliable, free hosting for the static frontend.
*   **gh-pages:** Deployment tool for automating the push to the production branch.
*   **Vite Copy Plugin:** Custom build step to generate `404.html` for client-side routing support.

## 5. Third-Party Integrations
*   **Menufy:** Backend provider for online ordering and menu management.

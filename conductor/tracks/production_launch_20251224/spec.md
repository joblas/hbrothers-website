# Track Spec: Production Launch and Handover Readiness

## 1. Goal
Finalize the H Brothers website for a production deployment that ranks #1 locally, performs flawlessly on mobile, and is ready for handover to the client.

## 2. Core Requirements
*   **SEO Verification:** Validate all meta tags, schema markup, and sitemap generation against the "Local Legend" strategy.
*   **Performance Audit:** Ensure Lighthouse scores are 90+ for Performance and Accessibility on mobile.
*   **AI Robustness:** Verify the AI Concierge fails gracefully when the API is unavailable and handles common restaurant queries accurately.
*   **Handover Documentation:** Create a clear guide for H Brothers on how to manage their API key and update the menu.

## 3. Success Metrics
*   Lighthouse Performance Score > 90.
*   Lighthouse Accessibility Score = 100.
*   Google Rich Results Test confirms valid "Restaurant" schema.
*   AI Concierge successfully answers: "What are your hours?", "Do you have brisket?", and "How do I order?".
